## 实践

maven项目可以直接依赖

```html
<dependency>
    <groupId>net.librec</groupId>
    <artifactId>librec-core</artifactId>
    <version>2.0.0</version>
</dependency>
```
为了更好的理解，这里直接拉取源码和对应的数据，跑通算例

```bash
git clone https://github.com/guoguibing/librec.git
```
